# SIEM Log Parser

A high-performance log parsing and alerting engine designed to detect security anomalies and potential authentication attacks in real-time.

## 🎯 Features
* **Log Correlation:** Extracts IP addresses and metadata from raw authentication syslog formats.
* **Threshold-Based Alerting:** Flags active brute force patterns when an IP exceeds defined failure rates.

## 🛡️ Applied Concepts
Demonstrates tactical blue-team operations, log structure normalization, and programmatic incident response detection.
